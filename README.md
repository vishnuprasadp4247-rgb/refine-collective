# Refine Collective

A modern and responsive business website designed to showcase a premium brand experience. Built with HTML, Tailwind CSS, and JavaScript, the project includes interactive navigation, gallery sections, contact forms, and a clean user interface optimized for all devices.

## Features
- Responsive Design
- Interactive Navigation
- Luxury Gallery Showcase
- Contact Form
- Modern UI/UX
- Mobile Friendly

## Technologies Used
- HTML5
- Tailwind CSS
- JavaScript
- Font Awesome
